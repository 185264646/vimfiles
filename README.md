# Personal vim config file

My personal vim config directory files

## Usage
### Unix
```bash
sudo apt install -y vim-youcompleteme
mkdir -p $HOME/.vim
cd $HOME/.vim
git init
git remote add origin git@github.com:185264646/vimfiles.git
git pull --set-upstream origin master
```

## Supported OS
- Windows 10+ (native/MSYS2)
- macOS
- common Linux distros (Debian series/Arch etc..)
- Android Termux
